Copy ShellRunAs.exe to the C:\Windows\System32 directory

Run ShellRunAs.reg

Run ShellRunAs.exe from your regular user account

Place PS1 file in folder on your PC and modify the paths within it to point to your MMC file

Place batch file with PS1 file and also modify the paths within to point to the right location (the PS1 file)

Place shortcut for the batch file on your Desktop

Right-click shortcut and click Run As Different User
